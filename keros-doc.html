/core

Core endpoints, used at multiple places in the application

/core/member get post

get

Gets a page of CoreTypes.Member. The response is under the form { "content": [ITEMS], "meta": {META} }

post

Creates an CoreTypes.Member and returns it

get /core/member

Gets a page of CoreTypes.Member. The response is under the form { "content": [ITEMS], "meta": {META} }

Secured by OAuth2-0

Simplified OAuth2.0 authentication in which every request must containt a JWT token header

  • Request
  • Response
  • Security

Query Parameters

  • poleId: (integer)

    If specified, add a search criteria to the users pole

  • positionId: (integer)

    If specified, add a search criteria to the users position

  • search: (string)

    Allows a search where any information of each item must match the searc term

    Example:

    thomas
  • orderBy: (string)

    The field by which to order by

    Example:

    name
  • order: (one of desc, asc - default: asc)

    Order

  • pageNumber: (integer - default: 0)

    The number of the page that is needed

  • pageSize: (integer - default: 25 - minimum: 10 - maximum: 100)

    Number of elements per page to recover
